# Robot Robby

A implementation of [Robby The Robot Genetic Algorithm](https://melaniemitchell.me/ExplorationsContent/RobbyTheRobot/) using haskell.

# Robby's world

## Possible actions

1. Move north
2. Move south
3. Move west
4. Move east
5. Stay put
6. Pick up can
7. Move random

## Possible slots state

1. Is empty
2. Has can
3. Is wall
